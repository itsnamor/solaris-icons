---
title: Global Styling
description: Advanced global styling techniques for Solaris Icons
---

By default, all icons have a **color** value of `currentColor` and **size** value of `1em`. Styling icons individually can be done by passing props to the icon component.

However, for consistent styling across your application, you can define global styles using CSS.

```css
.solaris {
  color: #4A90E2;    /* Set a global color for all icons */
  width: 24px;       /* Set a global size for all icons */
  height: 24px;      /* Set a global size for all icons */
}
```
